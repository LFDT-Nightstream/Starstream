# E0015: Duplicate Enum Variant

An enum definition contains the same variant name more than once.

## Example

```starstream
enum Status {
    Active,
    Inactive,
    Active,  // Error: variant `Active` appears multiple times in enum `Status`
}
```

## How to fix

Remove the duplicate variant or give it a unique name:

```starstream
enum Status {
    Active,
    Inactive,
    Pending,  // OK
}
```
