# E0051: `yield` outside `main fn`

Coroutine `yield` expressions may only appear inside coroutines functions, specifically `main fn`s inside `utxo`s.

## Example (error)

```starstream
fn example() {
    yield();
}
```

## Fix

Use separate `if ... is` blocks for each method call:

```starstream
abi ExampleAbi { }
utxo Example {
    impl ExampleAbi { }
    main fn example() {
        yield(ExampleAbi);
    }
}
```
