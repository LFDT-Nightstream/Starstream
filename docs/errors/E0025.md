# E0025: Match on Non-Enum Type

You tried to use `match` on a value that isn't an enum.

## Example

```starstream
fn main() {
    let x = 42;
    match x {  // Error: match scrutinee must be an enum, found `i64`
        // ...
    }
}
```

## How to fix

Use `match` only with enum values:

```starstream
enum Status {
    Active,
    Inactive,
}

fn describe(s: Status) -> i64 {
    match s {
        Status::Active => { 1 },
        Status::Inactive => { 0 },
    }
}
```

For non-enum values, use `if`/`else` expressions instead:

```starstream
fn describe(x: i64) -> i64 {
    if x > 0 {
        1
    } else {
        0
    }
}
```
