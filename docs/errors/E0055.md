# E0055: Reserved ABI name

`Token` is a built-in ABI provided by the compiler (it declares `attach` and
`detach` for `token` definitions). User code may not declare an `abi` with that
name.

## Example (error)

```starstream
abi Token {
    fn attach(to: Utxo);
}
```

## Fix

Choose a different ABI name:

```starstream
abi MyTokenAbi {
    fn attach(to: Utxo);
}
```
