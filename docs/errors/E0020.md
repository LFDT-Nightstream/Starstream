# E0020: Missing Struct Field

A struct literal is missing a required field.

## Example

```starstream
struct Point {
    x: i64,
    y: i64,
}

fn main() {
    let p = Point { x: 1 };  // Error: struct `Point` literal is missing field `y`
}
```

## How to fix

Provide all required fields:

```starstream
struct Point {
    x: i64,
    y: i64,
}

fn main() {
    let p = Point { x: 1, y: 2 };  // OK
}
```
